什么是smartJQ？

smartJQ 是 jQuery 的重构压缩版，在 jQuery API 的模型上，黑盒情况下制作的库；

兼容所有现代化的浏览器，因为不需要像jQuery一样对旧版本浏览器（IE10之前）兼容，所以体积更小；

应用场景

项目不考虑太旧的浏览器的兼容性（比如mobile web、整合性webview应用等），需要使用jQuery的场合，但又嫌弃jQuery体积太大的情况下，可以选用smartJQ；

相比zepto和JSLite实现jQuery半成品式的功能，smartJQ做到更完整的功能，

体积比zepto和JSLite都要小，并且具有更灵活方便的 自选rebuild 功能；
